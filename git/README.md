git Շպարգալկա
-------------

# git clone
```shell մեր ըքաութնում ռեպո ա  բացում ու քոփի անում մեր ըքաունդ ուռիշի կոդը
```
# git add 

```shell ափդեյթ ա անում 
```

# git commit
```զուտ սեյվ ա անում 
```

# git status
```ցույց ա տալիս վիճակը
```

# git push
```մեռ ռեպոյին ա ուղարկում մեր փոփոխած ֆայլերը
```

# git pull
```ռեպոյից հետ  ա քաշում
```
